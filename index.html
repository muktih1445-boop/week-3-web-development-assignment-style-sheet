About CSS

Cascading Style Sheets (CSS) is used to style and layout web pages. It controls colors, fonts, spacing, and overall presentation.

The Box Model

Every HTML element is treated as a box in CSS. The box consists of:

  • Content
  • Padding
  • Border
  • Margin
